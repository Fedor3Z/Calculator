# KinematicsCalc

Десктопное офлайн-приложение для расчёта «Расчет кинематики и системы нагружения».

## Возможности
- Расчёт по JSON-схеме формул без Excel.
- Оптимизация (SLSQP) с двухступенчатой стратегией.
- Проекты (.kproj), автосохранение, недавние проекты.
- Импорт входов из Excel (значения), экспорт отчётов в Excel и PDF.
- Русский UI, десятичная запятая, переключение темы и режима.

## Запуск
```bash
python -m app.main
```

## Сборка (Windows)
```bat
build\build.bat
```

Сборка создаст папку `dist\KinematicsCalc` с портативным приложением.
